# Vitality Global | Elite Digital Agency

Vitality Global is a high-performance landing page developed with Next.js 15, featuring a premium design and full multilingual support.

## 🚀 Teck Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Internationalization**: [next-intl v4](https://next-intl-docs.vercel.app/) (ES/EN support)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Heroicons](https://heroicons.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## ✨ Key Features

- **Multilingual Support**: Seamless switching between Spanish and English.
- **Next.js 15 Compatibility**: Handles asynchronous `params` and `getMessages` implementation.
- **Premium Design**: Dark mode aesthetic with glassmorphism and vibrant gradients.
- **Micro-animations**: Enhanced UX with Framer Motion transitions.
- **Responsive**: Fully optimized for mobile, tablet, and desktop devices.
- **High Performance**: Built with Next.js Turbopack for lightning-fast development and optimized production builds.

## 🛠️ Project Structure

- `src/app/[locale]`: Localized routes and core layout.
- `src/components`: Reusable UI components (Navbar, Hero, About, etc.).
- `src/messages`: Translation dictionaries (JSON).
- `src/i18n`: Internationalization configuration (Next.js 15 compatible).
- `src/navigation.ts`: Shared navigation utilities.

## 🏃 Getting Started

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
